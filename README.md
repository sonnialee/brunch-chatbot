# Brunch Chatbot

AI 챗봇으로 브런치 글 기반 커리어 조언 제공

## 기능

- 26개 브런치 글 기반 대화형 AI
- 이직, 이력서, PM 업무, AI 활용 관련 조언
- Claude API 기반 자연스러운 대화

## 기술 스택

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Claude API (Anthropic)
- Vercel 배포

## 설정

```bash
npm install
cp .env.example .env
# .env 파일에 ANTHROPIC_API_KEY 추가
npm run dev
```
